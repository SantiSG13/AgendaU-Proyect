# Agenda U

Documentación del proyecto Agenda U.

## Descripción
Sistema de agenda universitaria para la gestión de actividades académicas.

## Estructura del Proyecto
```
agenda-u/
├── index.html              ← Página de inicio (login)
├── registro.html
├── recuperarContraseña.html
├── dashboard.html
├── css/
│   ├── styles.css
│   └── dashboard.css
├── js/
│   └── main.js            ← (Opcional, para funcionalidad futura)
├── img/
│   └── logo.png
└── README.md              ← Documentación del proyecto
```

## Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript (opcional para funcionalidad futura)

## Instalación
1. Clonar el repositorio
2. Abrir `index.html` en un navegador web
